# Summary

This is a knob I made to replace two broken ones of an old Moulinex Microwave Oven.
Internal tube diameter is 6mm, external is 15mm.
Two version A and B with different slot angle (see pictures).

# Print Settings

Printer: Anet A8
Supports: Yes

Notes: 
I printed using supports but the internal one is difficult to remove, so try using just the outside support.
